# Employee Salary Processor API (In-Memory)

This project is a Spring Boot REST API that manages employees and performs salary-related operations using in-memory data (no database required).

## Features
- Add new employees
- Get employee by ID
- List all employees
- Delete employee
- Update employee salary
- Calculate average salary per department
- Find highest paid employee
- List employees earning above average
